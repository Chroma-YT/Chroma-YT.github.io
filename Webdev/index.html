Toggle Switch

This is a simple toggleable switch that has an output that can be detected by JavaScript, and has a customizable color spectrum. Click it to try it out!


Add this code to your files in the relevant locations.

index.html

<label class="toggle-switch">
  <input type="checkbox">
  <span class="slider"></span>
</label>

styles.css

.toggle-switch {
  position: relative;
  display: inline-block;
  width: 60px;
  height: 34px;
}

.toggle-switch input {
  opacity: 0;
  width: 0;
  height: 0;
}

.slider {
  position: absolute;
  cursor: pointer;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background-color: #ccc;
  transition: .4s;
  border-radius: 34px;
}

.slider:before {
  position: absolute;
  content: "";
  height: 26px;
  width: 26px;
  left: 4px;
  bottom: 4px;
  background-color: white;
  transition: .4s;
  border-radius: 50%;
}

input:checked + .slider {
  background-color: #2196F3;
}

input:checked + .slider:before {
  transform: translateX(26px);
}

.slider-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  width: 50%;
  margin: 0 auto;
}

.slider-description {
  flex: 1;
  text-align: left;
  padding-right: 10px;
}

script.js

const toggleSwitch = document.querySelector('.toggle-switch input');
let isSwitchOn = toggleSwitch.checked;

toggleSwitch.addEventListener('change', function() {
  isSwitchOn = toggleSwitch.checked;
  console.log(`Toggle Switch is ${isSwitchOn ? 'ON' : 'OFF'}`);
});

const rangeSlider = document.querySelector('.range-slider');
const rangeValue = document.querySelector('.range-value');
let sliderValue = rangeSlider.value;
rangeSlider.addEventListener('input', function() {
  sliderValue = rangeSlider.value;
  rangeValue.textContent = sliderValue;
  console.log(`Range Slider value: ${sliderValue}`);
});

const popupButton = document.querySelector('.popup-button');
const popup = document.querySelector('.popup');
popupButton.addEventListener('click', function() {
  popup.classList.add('show');
  setTimeout(function() {
    popup.classList.add('hide');
    setTimeout(function() {
      popup.classList.remove('show', 'hide');
    }, 500); // Wait for the glide out animation to complete
  }, 3000); // Popup will disappear after 3 seconds
});

Range Slider

This range slider allows you to select a value between 1 and 100. The selected value is displayed next to the slider and updates in real-time as you move the slider.

50


Add this code to your files in the relevant locations.

index.html

<div class="range-slider-container">
  <input type="range" min="1" max="100" value="50" class="range-slider">
  <span class="range-value">50</span>
</div>

</div>

Collapsible Sidebar

This button triggers a collapsible sidebar that can be used to display ads or other content.

× Ad Content Here


Add this code to your files in the relevant locations.

index.html

<button class="openbtn" onclick="openNav()">☰ Open Sidebar</button>
<div id="mySidebar" class="sidebar">
  <a href="javascript:void(0)" class="closebtn" onclick="closeNav()">×</a>
  <a href="#">Ad Content Here</a>
</div>

styles.css

.sidebar {
  height: 100%;
  width: 0;
  position: fixed;
  z-index: 1000;
  top: 0;
  right: 0;
  background-color: #333;
  overflow-x: hidden;
  transition: 0.5s;
  padding-top: 60px;
}

.sidebar a {
  padding: 10px 15px;
  text-decoration: none;
  font-size: 25px;
  color: #e0e0e0;
  display: block;
  transition: 0.3s;
}

.sidebar a:hover {
  color: #f1f1f1;
}

.sidebar .closebtn {
  position: absolute;
  top: 0;
  right: 25px;
  font-size: 36px;
  margin-left: 50px;
}

.openbtn {
  font-size: 20px;
  cursor: pointer;
  background-color: #444;
  color: #e0e0e0;
  border: none;
  padding: 10px 15px;
  border-radius: 3px;
  margin-top: 20px; /* Add space above the button */
}

.openbtn:hover {
  background-color: #555;
}

script.js

function openNav() {
  document.getElementById("mySidebar").style.width = "250px";
}

function closeNav() {
  document.getElementById("mySidebar").style.width = "0";
}
